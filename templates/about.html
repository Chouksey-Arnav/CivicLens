About CivicLens

CivicLens is an open-source tool that turns official congressional bill text into plain English and helps people find their representatives in Congress. It was built for the Congressional App Challenge.

What it does

Every bill introduced in Congress gets an official summary written by the Congressional Research Service. Those summaries are accurate, but they're written for people who already know legislative language. CivicLens takes that official summary and asks an AI model to rewrite it as a short, plain -English explainer: what the bill does, who it affects, its key provisions, and why it matters — without taking a side.

It also includes a "find your representatives" tool that looks up your current U.S. Senators and House member(s) directly from Congress.gov.

How accurate is the AI summary?

The AI is instructed to only use information from the official summary and to flag when something is unclear, but AI models can still make mistakes. Every bill page shows the original official summary right below the plain -English version, and links back to the full bill text and history on Congress.gov, so you can always check the source.

Run it yourself (free, local setup)

CivicLens runs on your own computer and uses two free APIs. Neither requires a credit card. This takes about 10 minutes the first time.

1. Get a free Congress.gov API key

Go to api.congress.gov/sign-up and sign up with your email. You'll get a key by email almost instantly. This key lets CivicLens pull real bill data (5,000 requests/hour, which is far more than one person needs).

2. Get a free Groq API key (for AI summaries)

Go to console.groq.com/keys and sign in with Google or email — no credit card needed. Create an API key. This powers the plain-English rewriting.

3. Install and configure

From a terminal, in the project folder:

pip install -r requirements.txt
cp .env.example .env

Open .env in any text editor and paste in your two keys: 

CONGRESS_API_KEY=your_key_here
GROQ_API_KEY=your_key_here

4. Run it

python api/app.py

Then open http://127.0.0.1:5000 in your browser.

Don't want to set up keys yet?

That's fine — CivicLens works right out of the box using bundled sample bills and sample representatives, clearly labeled as demo data, so you (or anyone reviewing this project) can explore the whole app with zero setup.

Data & tools used

  • Bill data: Congress.gov API (Library of Congress, free, public)
  • AI summaries: Groq API (free tier, Llama 3.3 model)
  • Backend: Python + Flask
  • Storage: a local SQLite file to cache AI summaries so they're only generated once
